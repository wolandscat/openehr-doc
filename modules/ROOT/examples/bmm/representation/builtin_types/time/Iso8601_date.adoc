3+^h|  *class Iso8601_date*

h|*Description*
2+a|Represents an ISO 8601 date, including partial and extended forms. Value may be:

* `YYYY-MM-DD` (extended, preferred)
* `YYYYMMDD` (compact)
* a partial invariant.

See `Time_definitions._valid_iso8601_date()` for validity.

h|*Inherit*
2+|xref:representation:representation-builtin_types.adoc#_class_iso8601_type[Iso8601_type]

h|
2+^h|*Feature Group "access"*

h| func
|*year* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Extract the year part of the date as an Integer.

h| func
|*month* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer] +
 +
  *pre_cond* +
    month_known: ¬ month_unknown;

a|Extract the month part of the date as an Integer, or return 0 if not present.

h| func
|*day* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer] +
 +
  *pre_cond* +
    day_known: ¬ day_unknown;

a|Extract the day part of the date as an Integer, or return 0 if not present.

h| func
|*timezone* : xref:representation:representation-builtin_types.adoc#_class_iso8601_timezone[Iso8601_timezone]
a|Timezone; may be Void.

h| func
|*month_unknown* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|Indicates whether month in year is unknown. If so, the date is of the form `"YYYY"`.

h| func
|*day_unknown* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|Indicates whether day in month is unknown. If so, and month is known, the date is of the
form `"YYYY-MM"` or `"YYYYMM"`.

h| func
|*as_string* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Return string value in extended format.
