3+^h|  *class Iso8601_duration*

h|*Description*
2+a|Represents an ISO 8601 duration, which may have multiple parts from years down to seconds.
The `value` attribute is a String in the format:

* `P[nnY][nnM][nnW][nnD][T[nnH][nnM][nnS]]`

NOTE: two deviations from ISO 8601 are supported, the first, to allow a negative sign, and
the second allowing the 'W' designator to be mixed with other designators.

h|*Inherit*
2+|xref:representation:builtin_types-gen.adoc#_class_iso8601_type[Iso8601_type]

h|
2+^h|*Feature Group "access"*

h| func
|*years* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of years in the `value`, i.e. the number preceding the `'Y'` in the `'YMD'` part,
if one exists.

h| func
|*months* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of months in the `value`, i.e. the value preceding the `'M'` in the `'YMD'` part,
if one exists.

h| func
|*weeks* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of weeks in the `value`, i.e. the value preceding the `W`, if one exists.

h| func
|*days* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of days in the `value`, i.e. the number preceding the `'D'` in the `'YMD'` part,
if one exists.

h| func
|*hours* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of hours in the `value`, i.e. the number preceding the `'H'` in the `'HMS'` part,
if one exists.

h| func
|*minutes* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of minutes in the `value`, i.e. the number preceding the `'M'` in the `'HMS'`
part, if one exists.

h| func
|*seconds* : xref:representation:builtin_types-gen.adoc#_class_integer[Integer]
a|Number of seconds in the `value`, i.e. the integer number preceding the `'S'` in the
`'HMS'` part, if one exists.

h| func
|*fractional_seconds* : xref:representation:builtin_types-gen.adoc#_class_real[Real]
a|Fractional seconds in the `value`, i.e. the decimal part of the number preceding the
`'S'` in the `'HMS'` part, if one exists.

h| func
|*to_seconds* : xref:representation:builtin_types-gen.adoc#_class_real[Real]
a|Total number of seconds equivalent (including fractional) of entire duration. Where non-
definite elements such as year and month (i.e. 'Y' and 'M') are included, the
corresponding 'average' durations from `Time_definitions` are used to compute the result.

h| func
|*as_string* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Return the duration string value.

2+a|

h|

h|
2+a|days_valid: days >= 0;

h|
2+a|hours_valid: hours >= 0;

h|
2+a|minutes_valid: minutes >= 0;

h|
2+a|seconds_valid: seconds >= 0;

h|
