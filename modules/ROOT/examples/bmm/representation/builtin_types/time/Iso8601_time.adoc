3+^h|  *class Iso8601_time*

h|*Description*
2+a|Represents an ISO 8601 time, including partial and extended forms. Value may be
hh:mm:ss (extended, preferred), hhmmss (compact), or a partial invariant.

h|*Inherit*
2+|xref:representation:representation-builtin_types.adoc#_class_iso8601_type[Iso8601_type]

h|
2+^h|*Feature Group "access"*

h| func
|*hour* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Extract the hour part of the time as an Integer.

h| func
|*minute* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Extract the minute part of the time as an Integer, or return 0 if not present.

h| func
|*second* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Extract the integral seconds part of the time as an Integer, or return 0 if
not present.

h| func
|*fractional_second* : xref:representation:representation-builtin_types.adoc#_class_real[Real]
a|Extract the fractional seconds part of the time as a Real.

h| func
|*timezone* : xref:representation:representation-builtin_types.adoc#_class_iso8601_timezone[Iso8601_timezone]
a|Timezone; may be Void.

h| func
|*minute_unknown* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|Indicates whether minute is unknown.

h| func
|*second_unknown* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|Indicates whether second is unknown.

h| func
|*as_string* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Return string value in extended format.
