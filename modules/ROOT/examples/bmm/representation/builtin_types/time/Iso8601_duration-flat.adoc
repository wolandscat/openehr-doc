3+^h|  *class Iso8601_duration*

h|*Description*
2+a|Represents an ISO 8601 duration, which may have multiple parts from years down
to seconds. Value format: P[nnY][nnM][nnW][nnD][T[nnH][nnM][nnS]].

h|*Inherit*
2+|xref:representation:representation-builtin_types.adoc#_class_iso8601_type[Iso8601_type]

h|
2+^h|*Feature Group "access"*

h| func
|*years* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of years in the `_value_`, i.e. the number preceding the `'Y'` in the `'YMD'` part,
if one exists.

h| func
|*months* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of months in the `_value_`, i.e. the value preceding the `'M'` in the `'YMD'` part,
if one exists.

h| func
|*weeks* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of weeks in the `_value_`, i.e. the value preceding the `W`, if one exists.

h| func
|*days* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of days in the `_value_`, i.e. the number preceding the `'D'` in the `'YMD'` part,
if one exists.

h| func
|*hours* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of hours in the `_value_`, i.e. the number preceding the `'H'` in the `'HMS'` part,
if one exists.

h| func
|*minutes* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of minutes in the `_value_`, i.e. the number preceding the `'M'` in the `'HMS'`
part, if one exists.

h| func
|*seconds* : xref:representation:representation-builtin_types.adoc#_class_integer[Integer]
a|Number of seconds in the `_value_`, i.e. the integer number preceding the `'S'` in the
`'HMS'` part, if one exists.

h| func
|*fractional_seconds* : xref:representation:representation-builtin_types.adoc#_class_real[Real]
a|Fractional seconds in the `_value_`, i.e. the decimal part of the number preceding the
`'S'` in the `'HMS'` part, if one exists.

h| func
|*to_seconds* : xref:representation:representation-builtin_types.adoc#_class_real[Real]
a|Total number of seconds equivalent (including fractional) of entire duration. Where non-
definite elements such as year and month (i.e. 'Y' and 'M') are included, the
corresponding 'average' durations from `Time_definitions` are used to compute the result.

h| func
|*as_string* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Return the duration string value.

h|prop +
  
|*value*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Representation of all descendants is a single String.

h| func
|*is_partial* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this date time is partial, i.e. if trailing end (right hand) value(s) is/are
missing.

h| func
|*is_extended* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this ISO8601 string is in the 'extended' form, i.e. uses `'-'` and / or `':'`
separators. This is the preferred format.
