3+^h|  *class Cardinality*

h|*Description*
2+a|Express constraints on the cardinality of container objects which are the
values of multiply-valued attributes, including uniqueness and ordering, providing
the means to state that a container acts like a logical list, set or bag.

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*interval*: xref:representation:representation-builtin_types.adoc#_class_multiplicity_interval[Multiplicity_Interval]
a|The interval of this cardinality.

h|prop +
  
|*is_ordered*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the members of the container attribute are ordered.

h|prop +
  
|*is_unique*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the members of the container attribute are unique.

h| func
|*is_bag* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the semantics represent a bag (unordered, non-unique membership).

h| func
|*is_list* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the semantics represent a list (ordered, non-unique membership).

h| func
|*is_set* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the semantics represent a set (unordered, unique membership).
