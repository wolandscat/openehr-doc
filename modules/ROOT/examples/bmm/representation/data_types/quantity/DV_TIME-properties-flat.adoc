3+^h|  *class DV_TIME*

h|*Description*
2+a|Represents an absolute point in time from an origin usually interpreted as meaning the
start of the current day, specified to a fraction of a second. Semantics defined by ISO
8601.

Used for recording real world times, rather than scientifically measured fine amounts of
time. The partial form is used for approximate times of events and substance
administrations.

|*magnitude_status*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Optional status of magnitude with values:

* `"="` : magnitude is a point value
* `"<"` : value is < magnitude
* `">"` : value is > magnitude
* `"<="` : value is <= magnitude
* `">="` : value is >= magnitude
* `"~"` : value is approximately magnitude

If not present, assumed meaning is `"="` .
