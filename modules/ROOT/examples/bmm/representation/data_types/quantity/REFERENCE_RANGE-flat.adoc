h|*Description*
2+a|Defines a named range to be associated with any DV_ORDERED datum. May represent normal, therapeutic,
dangerous, critical ranges.

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*meaning*: xref:representation:representation-data_types.adoc#_class_dv_text[DV_TEXT]
a|Term whose value indicates the meaning of this range, e.g. normal, critical, therapeutic
etc.

h|prop +
  
|*range*: xref:representation:representation-data_types.adoc#_class_dv_interval[DV_INTERVAL]<T>
a|The data range for this meaning, e.g. critical etc.

h|*Invariants*
2+a|

h|
