3+^h|  *class DV_SCALE*


h|
2+^h|*Feature Group "access"*

h|prop +
  
|*symbol*: xref:representation:data_types-gen.adoc#_class_dv_coded_text[DV_CODED_TEXT]
a|Coded textual representation of this value in the scale range, which may be strings made
from symbols or other enumerations of terms such as `no breathlessness`, `very very
slight`, `slight breathlessness`. Codes come from archetypes.

In some cases, a scale may include values that have no code/symbol. In this case, the
symbol will be a `DV-CODED_TEXT` including the `terminology_id` and a blank String value
for `code_string`.

h|prop +
  
|*value*: xref:representation:builtin_types-gen.adoc#_class_real[Real]
a|Real number value of Scale item.

h|prop +
  0..1
|*normal_status*: xref:representation:data_types-gen.adoc#_class_code_phrase[CODE_PHRASE]
a|Optional normal status indicator of value with respect to normal range for this value.
Often included by lab, even if the normal range itself is not included. Coded by ordinals
in series HHH, HH, H, (nothing), L, LL, LLL; see openEHR terminology group
`normal_status`.

h|prop +
  0..1
|*normal_range*: xref:representation:data_types-gen.adoc#_class_dv_interval[DV_INTERVAL]<xref:representation:data_types-gen.adoc#_class_dv_ordered[DV_ORDERED]>
a|Optional normal range.

h|prop +
  0..1

a|Optional tagged other reference ranges for this value in its particular measurement
context.
