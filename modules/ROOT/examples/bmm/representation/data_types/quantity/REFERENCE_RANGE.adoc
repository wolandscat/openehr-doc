3+^h|  *class REFERENCE_RANGE<T>*

h|*Description*
2+a|Defines a named range to be associated with any `DV_ORDERED` datum. Each such range is
particular to the patient and context, e.g. sex, age, and any other factor which affects
ranges. May be used to represent normal, therapeutic, dangerous, critical etc ranges.

h|*Inherit*
2+|xref:representation:builtin_types-gen.adoc#_class_any[Any]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*meaning*: xref:representation:data_types-gen.adoc#_class_dv_text[DV_TEXT]
a|Term whose value indicates the meaning of this range, e.g. normal, critical, therapeutic
etc.

h|prop +
  
|*range*: xref:representation:data_types-gen.adoc#_class_dv_interval[DV_INTERVAL]<T>
a|The data range for this meaning, e.g. critical etc.

h|*Invariants*
2+a|

h|
2+a|range_is_simple: (range.lower_unbounded ∨ range.lower.is_simple) ∧ (range.upper_unbounded ∨ range.upper.is_simple);
