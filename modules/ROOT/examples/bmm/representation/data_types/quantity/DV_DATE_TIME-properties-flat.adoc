3+^h|  *class DV_DATE_TIME*

h|*Description*
2+a|Represents an absolute point in time, specified to the second. Semantics defined by ISO
8601.

Used for recording a precise point in real world time, and for approximate time stamps,
e.g. the origin of a `HISTORY` in an `OBSERVATION` which is only partially known.

|*magnitude_status*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Optional status of magnitude with values:

* `"="` : magnitude is a point value
* `"<"` : value is < magnitude
* `">"` : value is > magnitude
* `"<="` : value is <= magnitude
* `">="` : value is >= magnitude
* `"~"` : value is approximately magnitude

If not present, assumed meaning is `"="` .

in series HHH, HH, H, (nothing), L, LL, LLL; see openEHR terminology group
`normal_status`.
