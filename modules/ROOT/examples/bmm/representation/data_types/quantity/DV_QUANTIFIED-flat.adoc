|*magnitude_status*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Optional status of magnitude with values:

* `"="` : magnitude is a point value
* `"<"` : value is < magnitude
* `">"` : value is > magnitude
* `"<="` : value is <= magnitude
* `">="` : value is >= magnitude
* `"~"` : value is approximately magnitude

If not present, assumed meaning is `"="` .
