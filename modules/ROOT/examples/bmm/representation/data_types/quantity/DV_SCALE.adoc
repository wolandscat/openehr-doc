3+^h|  *class DV_SCALE*

h|*Description*
2+a|A data type that represents scale values with implied ordering, no implication of constant distance,
finite total values, allowing non-integer values.

h|*Inherit*
2+|xref:representation:representation-data_types.adoc#_class_dv_ordered[DV_ORDERED]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*symbol*: xref:representation:representation-data_types.adoc#_class_dv_coded_text[DV_CODED_TEXT]
a|Coded textual representation of this value in the scale range, which may be strings made
from symbols or other enumerations of terms such as `no breathlessness`, `very very
slight`, `slight breathlessness`. Codes come from archetypes.

In some cases, a scale may include values that have no code/symbol. In this case, the
symbol will be a `DV-CODED_TEXT` including the `_terminology_id_` and a blank String value
for `_code_string_`.

h|prop +
  
|*value*: xref:representation:representation-builtin_types.adoc#_class_real[Real]
a|Real number value of Scale item.
