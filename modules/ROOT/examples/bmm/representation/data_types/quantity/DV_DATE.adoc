3+^h|  *class DV_DATE*

h|*Description*
2+a|Represents an absolute point in time, as measured on the Gregorian calendar, and specified
only to the day. Semantics defined by ISO 8601. Used for recording dates in real world
time. The partial form is used for approximate birth dates, dates of death, etc.
