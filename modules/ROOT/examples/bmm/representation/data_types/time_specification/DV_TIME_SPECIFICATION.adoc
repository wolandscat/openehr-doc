3+^h| abstract  *class DV_TIME_SPECIFICATION*

h|*Description*
2+a|This is an abstract class of which all timing specifications are specialisations.
Specifies points in time, possibly linked to the calendar, or a real world repeating
event, such as breakfast.

h|*Inherit*
2+|xref:representation:data_types-gen.adoc#_class_data_value[DATA_VALUE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*value*: xref:representation:data_types-gen.adoc#_class_dv_parsable[DV_PARSABLE]
a|The specification, in the HL7v3 syntax for `PIVL` or `EIVL` types.

h| func
|*calendar_alignment* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Indicates what prototypical point in the calendar the specification is aligned to, e.g.
5th of the month . Empty if not aligned. Extracted from the value' attribute.

h| func
|*event_alignment* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Indicates what real-world event the specification is aligned to if any. Extracted from the
value' attribute.

h| func
|*institution_specified* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|Indicates if the specification is aligned with institution schedules, e.g. a hospital
nursing changeover or meal serving times. Extracted from the value' attribute.
