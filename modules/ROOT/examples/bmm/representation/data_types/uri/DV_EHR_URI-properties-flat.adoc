3+^h|  *class DV_EHR_URI*

h|*Description*
2+a|A `DV_EHR_URI` is a `DV_URI` which has the scheme name 'ehr', and which can only reference
items in EHRs.

Used to reference items in an EHR, which may be the same as the current EHR (containing
this link), or another.

h|*Inherit*
2+|xref:representation:representation-data_types.adoc#_class_dv_uri[DV_URI]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*value*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Value of URI as a String. 'Plain-text' URIs are allowed, enabling better readability, but
must be RFC-3986 encoded in use.

h|*Invariants*
2+a|

h|
2+a|scheme_valid: scheme.is_equal(Ehr_scheme);
