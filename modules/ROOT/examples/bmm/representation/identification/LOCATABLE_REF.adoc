3+^h|  *class LOCATABLE_REF*

h|*Description*
2+a|Reference to a LOCATABLE instance inside the top-level content structure inside a VERSION<T>.

h|*Inherit*
2+|xref:representation:representation-identification.adoc#_class_object_ref[OBJECT_REF]

h|
2+^h|*Feature Group "access"*

h|prop +
  0..1
|*path*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|The path to an instance, as an absolute path with respect to the object found at
`VERSION._data_`. An empty path means that the object referred to by `_id_` is being
specified.

h|prop +
  
|*id*: xref:representation:representation-identification.adoc#_class_uid_based_id[UID_BASED_ID]
a|Globally unique id of an object, regardless of where it is stored.

h| func
|*as_uri* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|A URI form of the reference, created by concatenating the following:

* scheme, e.g. `ehr:`, derived from `_namespace_`
* `_id.value_`
* `/` + `_path_`, where `_path_` is non-empty
