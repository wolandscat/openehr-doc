3+^h|  *class OBJECT_VERSION_ID*

h|*Description*
2+a|Globally unique identifier for one version of a versioned object. Lexical form:
object_id '::' creating_system_id '::' version_tree_id.

h|*Inherit*
2+|xref:representation:representation-identification.adoc#_class_uid_based_id[UID_BASED_ID]

h|
2+^h|*Feature Group "access"*

h| func
|*object_id* : xref:representation:representation-identification.adoc#_class_uid[UID]
a|Unique identifier for logical object of which this identifier identifies one version;
normally the `_object_id_` will be the unique identifier of the version container
containing the version referred to by this `OBJECT_VERSION_ID` instance.

h| func
|*creating_system_id* : xref:representation:representation-identification.adoc#_class_uid[UID]
a|Identifier of the system that created the Version corresponding to this Object version id.

h| func
|*version_tree_id* : xref:representation:representation-identification.adoc#_class_version_tree_id[VERSION_TREE_ID]
a|Tree identifier of this version with respect to other versions in the same version tree,
as either 1 or 3 part dot-separated numbers, e.g. 1 , 2.1.4 .

h| func
|*is_branch* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this version identifier represents a branch.

h| func
|*root* : xref:representation:representation-identification.adoc#_class_uid[UID]
a|The identifier of the conceptual namespace in which the object exists, within the
identification scheme. Returns the part to the left of the first '::' separator, if any,
or else the whole string.

h| func
|*extension* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Optional local identifier of the object within the context of the root identifier. Returns
the part to the right of the first '::' separator if any, or else any empty String.

h| func
|*has_extension* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if not `_extension_.is_empty()`.

h|prop +
  
|*value*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|The value of the id in the form defined below.
