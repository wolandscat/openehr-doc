3+^h|  *class ARCHETYPE_ID*

h|*Description*
2+a|Identifier for archetypes. Lexical form: rm_originator '-' rm_name '-' rm_entity '.' concept_name
{'-' specialisation}* '.v' version_id.

h|*Inherit*
2+|xref:representation:representation-identification.adoc#_class_object_id[OBJECT_ID]

h|
2+^h|*Feature Group "access"*

h| func
|*qualified_rm_entity* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Globally qualified reference model entity, e.g. `"openehr-EHR-OBSERVATION"`.

h| func
|*domain_concept* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Name of the concept represented by this archetype, including specialisation, e.g.
`"Biochemistry_result-cholesterol"`.

h| func
|*rm_originator* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Organisation originating the reference model on which this archetype is based, e.g.
`"openEHR"`, `"CEN"`, `"HL7"`.

h| func
|*rm_name* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Name of the reference model, e.g. `"RIM"`, `"EHR"`, `"EN13606"`.

h| func
|*rm_entity* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Name of the ontological level within the reference model to which this archetype is
targeted, e.g. for openEHR: `"FOLDER"`, `"COMPOSITION"`, `"SECTION"`, `"OBSERVATION"`.

h| func
|*specialisation* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Name of specialisation of concept, if this archetype is a specialisation of another
archetype, e.g. `"cholesterol"`.

h| func
|*version_id* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Version identification of this archetype, representing the major version, e.g. `"1"`.
