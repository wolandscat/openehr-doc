3+^h|  *class VERSION_TREE_ID*

h|*Description*
2+a|Version tree identifier for one version. Lexical form: trunk_version ['.' branch_number '.' branch_version].

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*value*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|String form of this identifier.

h| func
|*trunk_version* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Trunk version number; numbering starts at 1.

h| func
|*is_branch* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this version identifier represents a branch, i.e. has `_branch_number()_` and
`_branch_version()_` parts.

h| func
|*is_first* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this version identifier corresponds to the first version, i.e. `_trunk_version_`
is 1.

h| func
|*branch_number* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Number of branch from the trunk point; numbering starts at 1.

h| func
|*branch_version* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Version of the branch; numbering starts at 1.

h|*Invariants*
2+a|

h|
2+a|value_valid: ¬ value.is_empty();

h|
2+a|trunk_version_valid: ∃ trunk_version ∧ trunk_version.is_integer ∧ trunk_version.as_integer >= 1;

h|
2+a|branch_number_valid: ∃ branch_number ⇒ branch_number.is_integer ∧ branch_number.as_integer >= 1;

h|
2+a|branch_version_valid: ∃ branch_version ⇒ branch_version.is_integer ∧ branch_version.as_integer >= 1;

h|
2+a|branch_validity: (¬ ∃ branch_number ∧ ¬ ∃ branch_version) ⊻ (∃ branch_number ∧ ∃ branch_version);

h|
2+a|is_branch_validity: is_branch ⊻ ¬ ∃ branch_number;

h|
2+a|is_first_validity: ¬ is_first ⊻ trunk_version.is_equal("1");
