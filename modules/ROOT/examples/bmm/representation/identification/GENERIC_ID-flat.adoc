h|*Description*
2+a|Generic identifier type for identifiers whose format is otherwise unknown to openEHR.
Includes an attribute for naming the identification scheme (which may well be local).

h|*Inherit*
2+|xref:representation:identification-gen.adoc#_class_object_id[OBJECT_ID]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*scheme*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Name of the scheme to which this identifier conforms. Ideally this name will be
recognisable globally but realistically it may be a local ad hoc scheme whose name is not
controlled or standardised in any way.

|*value*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|The value of the id in the form defined below.
