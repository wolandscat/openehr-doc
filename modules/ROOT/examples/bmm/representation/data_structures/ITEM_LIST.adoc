3+^h|  *class ITEM_LIST*

h|*Description*
2+a|Logical list data structure, where each item has a value and can be referred to by a name
and a positional index in the list. The list may be empty.

`ITEM_LIST` is used to represent any data which is logically a list of values, such as
blood pressure, most protocols, many blood tests etc.

Not to be used for time-based lists, which should be represented with the proper temporal
class, i.e. `HISTORY`.
