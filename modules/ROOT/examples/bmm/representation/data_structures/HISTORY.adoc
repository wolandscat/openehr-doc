3+^h|  *class HISTORY<T:ITEM_STRUCTURE>*

h|*Description*
2+a|Root object of a linear history, i.e. time series structure. This is a generic class whose
type parameter must be a descendant of `ITEM_STRUCTURE`, ensuring that each Event in the
`events` of a given instance is of the same structural type, i.e. `ITEM_TREE`,
`ITEM_LIST` etc.

For a periodic series of events, period will be set, and the time of each Event in the
History must correspond; i.e. the `EVENT._offset` must be a multiple of period for each
Event. Missing events in a period History are however allowed.

h|*Inherit*
2+|xref:representation:data_structures-gen.adoc#_class_data_structure[DATA_STRUCTURE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*origin*: xref:representation:data_types-gen.adoc#_class_dv_date_time[DV_DATE_TIME]
a|Time origin of this event history. The first event is not necessarily at the origin point.

h|prop +
  0..1
|*period*: xref:representation:data_types-gen.adoc#_class_dv_duration[DV_DURATION]
a|Period between samples in this segment if periodic.

h|prop +
  0..1
|*duration*: xref:representation:data_types-gen.adoc#_class_dv_duration[DV_DURATION]
a|Duration of the entire History; either corresponds to the duration of all the events,
and/or the duration represented by the summary, if it exists.

h|prop +
  0..1
|*summary*: xref:representation:data_structures-gen.adoc#_class_item_structure[ITEM_STRUCTURE]
a|Optional summary data that aggregates, organizes, reduces and transforms the event series.
This may be a text or image that presents a graphical presentation, or some data that
assists with the interpretation of the data.

h|prop +
  0..1
|*events*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:data_structures-gen.adoc#_class_event[EVENT]<T>>
a|The events in the series. This attribute is of a generic type whose parameter must be a
descendant of `ITEM_SUTRUCTURE`.

h| func
|*is_periodic* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|Indicates whether history is periodic.
