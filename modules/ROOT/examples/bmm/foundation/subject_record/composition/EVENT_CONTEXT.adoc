3+^h|  *class EVENT_CONTEXT*

h|*Description*
2+a|Documents the context information of a healthcare event involving the subject of care
and the health system. The context information recorded here is independent of the
attributes recorded in the version audit, which document the system interaction
context, i.e. the context of a user interacting with the health record system.

h|*Inherit*
2+|xref:representation:representation-data_marking.adoc#_class_pathable[PATHABLE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*start_time*: xref:representation:representation-data_types.adoc#_class_dv_date_time[DV_DATE_TIME]
a|Start time of the clinical session or other kind of event during which a provider
performs a service of any kind for the patient.

h|prop +
  0..1
|*end_time*: xref:representation:representation-data_types.adoc#_class_dv_date_time[DV_DATE_TIME]
a|Optional end time of the clinical session.

h|prop +
  0..1
|*location*: xref:representation:representation-builtin_types.adoc#_class_string[String]

ward A3.

|*setting*: xref:representation:representation-data_types.adoc#_class_dv_coded_text[DV_CODED_TEXT]

h|prop +
  0..1
|*other_context*: xref:representation:representation-data_structures.adoc#_class_item_structure[ITEM_STRUCTURE]

h|prop +
  0..1
|*health_care_facility*: xref:representation:representation-data_marking.adoc#_class_party_identified[PARTY_IDENTIFIED]

  0..1
