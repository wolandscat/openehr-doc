3+^h|  *class ACTION*

h|*Description*
2+a|Used to record a clinical action that has been performed, which may have been ad hoc,
or due to the execution of an Activity in an Instruction workflow. Every Action
corresponds to a careflow step of some kind or another.

h|*Inherit*
2+|xref:foundation:foundation-subject_record.adoc#_class_care_entry[CARE_ENTRY]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*time*: xref:representation:representation-data_types.adoc#_class_dv_date_time[DV_DATE_TIME]
a|Point in time at which this action completed. To indicate an unknown time, use a
`DV_DATE_TIME` instance with `_value_` set to the time of creation (or some other known
time before which the Action is known to have occurred, e.g. data accession timestamp from
integration engine), and `_magnitude_status_` set to `<`.

h|prop +
  
|*ism_transition*: xref:foundation:foundation-subject_record.adoc#_class_ism_transition[ISM_TRANSITION]
a|Details of transition in the Instruction state machine caused by this Action.

h|prop +
  0..1
|*instruction_details*: xref:foundation:foundation-subject_record.adoc#_class_instruction_details[INSTRUCTION_DETAILS]
a|Details of the Instruction that caused this Action to be performed, if there was one.

h|prop +
  
|*description*: xref:representation:representation-data_structures.adoc#_class_item_structure[ITEM_STRUCTURE]
a|Description of the action that has been performed, in the form of an archetyped structure.
