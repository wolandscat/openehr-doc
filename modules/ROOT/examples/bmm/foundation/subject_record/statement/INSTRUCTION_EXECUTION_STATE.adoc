2+a|Enumeration of states in the Instruction State Machine (ISM) used to track the

2+|xref:representation:builtin_types-gen.adoc#_class_string[String]

2+^h|*Feature Group "values"*

h|
|*initial* = initial
a|
Instruction created but not yet activated.



h|
|*planned* = planned
a|
Instruction has been planned.



h|
|*postponed* = postponed
a|
Instruction execution has been postponed.



h|
|*cancelled* = cancelled
a|
Instruction has been cancelled prior to start.



h|
|*active* = active
a|
Instruction is currently being executed.



h|
|*suspended* = suspended
a|
Instruction execution has been suspended.



h|
|*completed* = completed
a|
Instruction has been completed.



h|
|*aborted* = aborted
a|
Instruction execution has been aborted.



h|
|*expired* = expired
a|
Instruction has expired without completion.
