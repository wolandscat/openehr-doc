3+^h|  *class OBSERVATION*

h|*Description*
2+a|Entry subtype for all clinical data in the past or present, i.e. which (by the time
it is recorded) has already occurred. OBSERVATION data is expressed using the class
HISTORY<T>, which guarantees that it is situated in time. OBSERVATION is used for all
notionally objective (i.e. measured in some way) observations of phenomena, and
patient-reported phenomena, e.g. pain.

h|*Inherit*
