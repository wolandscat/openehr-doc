3+^h|  *class ROLE*

h|*Description*
2+a|Generic description of a role performed by an Actor. The role corresponds to a
competency of the Party. Roles are used to define the responsibilities undertaken
by a Party for a purpose. Roles should have credentials qualifying the performer to
perform the role.
