3+^h| abstract  *class EXTRACT_CONTENT_ITEM*

h|*Description*
2+a|Abstract model of a wrapper for one content item in an Extract, containing various
meta-data. Indicates whether it was part of the primary set and what its original
path was. Intended to be subtyped for wrappers of specific types of content.

h|*Inherit*
2+|xref:domain:domain-healthcare-ehr_extract.adoc#_class_extract_item[EXTRACT_ITEM]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*is_primary*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content item carried in this container was part of the primary set
for the Extract, i.e. not added due to link-following.

h|prop +
  0..1
|*is_changed*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content item carried in this container is any kind of change since
last send, in repeat sending situations.

h|prop +
  0..1
|*is_masked*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content of this item has not been included due to insufficient access
rights of requestor.

h|prop +
  0..1
|*item*: xref:representation:representation-data_marking.adoc#_class_locatable[LOCATABLE]
a|Content object.

h|prop +
  
|*name*: xref:representation:representation-data_types.adoc#_class_dv_text[DV_TEXT]
a|Runtime name of this fragment, used to build runtime paths.

h|prop +
  0..1
|*links*: xref:representation:representation-builtin_types.adoc#_class_list[List]<xref:representation:representation-data_marking.adoc#_class_link[LINK]>
a|Links to other archetyped structures.

h|prop +
  0..1
|*feeder_audit*: xref:representation:representation-data_marking.adoc#_class_feeder_audit[FEEDER_AUDIT]
a|Audit trail from non-openEHR system of original commit of information.

h|
2+^h|*Feature Group "infrastructure"*

h|prop +
  0..1
|*uid*: xref:representation:representation-identification.adoc#_class_uid_based_id[UID_BASED_ID]
a|Optional globally unique object identifier for root points of archetyped structures.

h|prop +
  
|*archetype_node_id*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Design-time archetype identifier of this node, used to build archetype paths.

h|prop +
  0..1
|*archetype_details*: xref:representation:representation-data_marking.adoc#_class_archetyped[ARCHETYPED]
a|Details of archetyping used on this node.
