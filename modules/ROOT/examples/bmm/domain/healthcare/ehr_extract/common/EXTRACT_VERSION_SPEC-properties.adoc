3+^h|  *class EXTRACT_VERSION_SPEC*

h|*Description*
2+a|Specification of what versions should be included in an Extract. By default, only
latest versions are included. The include_all_versions and commit_time_interval
attributes are used to modify this behaviour.

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*include_all_versions*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if all versions of each item in the Extract are included.

h|prop +
  0..1
|*commit_time_interval*: xref:representation:representation-data_types.adoc#_class_dv_interval[DV_INTERVAL]<xref:representation:representation-data_types.adoc#_class_dv_date_time[DV_DATE_TIME]>
a|Specifies commit time interval of items to source repository to include in
Extract.

h|prop +
  
|*include_revision_history*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if revision histories of the items in the Extract are included. If included,
it is always the full revision history.

h|prop +
  
|*include_data*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the data of items matched by the content spec should be included. If
False, only revision history is included.
