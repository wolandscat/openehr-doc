3+^h| abstract  *class EXTRACT_CONTENT_ITEM*

h|*Description*
2+a|Abstract model of a wrapper for one content item in an Extract, containing various meta-
data. Indicates whether it was part of the primary set and what its original path was.
Intended to be subtyped for wrappers of specific types of content.

h|*Inherit*
2+|xref:domain:domain-healthcare-ehr_extract.adoc#_class_extract_item[EXTRACT_ITEM]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*is_primary*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content item carried in this container was part of the primary set for the
Extract, i.e. not added due to link-following.

h|prop +
  0..1
|*is_changed*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content item carried in this container is any kind of change since last send,
in repeat sending situations.

h|prop +
  0..1
|*is_masked*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the content of this item has not been included due to insufficient access rights
of requestor.

h|prop +
  0..1
|*item*: xref:representation:representation-data_marking.adoc#_class_locatable[LOCATABLE]
a|Content object.

h|*Invariants*
2+a|

h|
2+a|item_validity: is_masked ⊻ ∃ item;
