3+^h|  *class EHR_STATUS*

h|*Description*
2+a|Single object per EHR containing various EHR-wide status flags and settings,
including whether this EHR can be queried, modified etc. This object is always
modifiable, in order to change the status of the EHR as a whole.

h|*Inherit*
2+|xref:representation:representation-data_marking.adoc#_class_locatable[LOCATABLE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*subject*: xref:representation:representation-data_marking.adoc#_class_party_self[PARTY_SELF]
a|The subject of this EHR. The external_ref attribute can be used to contain a
direct reference to the subject in a demographic or identity service.

h|prop +
  
|*is_queryable*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if this EHR should be included in population queries, i.e. if this EHR is
considered active in the population.

h|prop +
  
|*is_modifiable*: xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
a|True if the EHR, other than the EHR_STATUS object, is allowed to be written to.

h|prop +
  0..1
|*other_details*: xref:representation:representation-data_structures.adoc#_class_item_structure[ITEM_STRUCTURE]
a|Any other details of the EHR summary object, in the form of an archetyped
ITEM_STRUCTURE.
