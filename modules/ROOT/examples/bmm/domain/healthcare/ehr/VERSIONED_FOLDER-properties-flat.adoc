3+^h|  *class VERSIONED_FOLDER*

h|*Description*
2+a|A version-controlled hierarchy of `FOLDERs` giving the effect of a directory.

h|*Inherit*
2+|xref:infostructure:infostructure-versioning.adoc#_class_versioned_object[VERSIONED_OBJECT]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*uid*: xref:representation:representation-identification.adoc#_class_hier_object_id[HIER_OBJECT_ID]
a|Unique identifier of this version container in the form of a UID with no extension. This
id will be the same in all instances of the same container in a distributed environment,
meaning that it can be understood as the uid of the virtual version tree.

h|prop +
  
|*owner_id*: xref:representation:representation-identification.adoc#_class_object_ref[OBJECT_REF]
a|Reference to object to which this version container belongs, e.g. the id of the containing
EHR or other relevant owning entity.

h|prop +
  
|*time_created*: xref:representation:representation-data_types.adoc#_class_dv_date_time[DV_DATE_TIME]
a|Time of initial creation of this versioned object.
