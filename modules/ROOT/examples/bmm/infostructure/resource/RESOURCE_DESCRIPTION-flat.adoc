3+^h|  *class RESOURCE_DESCRIPTION*

h|*Description*
2+a|Defines the descriptive meta-data of a resource.

h|
2+^h|*Feature Group "access"*

h|prop +
  0..1
|*title*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Long-form descriptive title of the resource, typically needed in templates and
specialised archetypes where the root name alone is insufficient.

h|prop +
  
|*original_author*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|Original author of this resource, with all relevant details including organisation.

h|prop +
  0..1
|*original_namespace*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Namespace of original author's organisation, in reverse internet form, if applicable.

h|prop +
  0..1
|*original_publisher*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Plain text name of organisation that originally published this artefact, if any.

h|prop +
  0..1
|*other_contributors*: xref:representation:representation-builtin_types.adoc#_class_list[List]<xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|Other contributors to the resource, each listed in "name <email>" form.

h|prop +
  
|*parent_resource*: xref:infostructure:infostructure-resource.adoc#_class_authored_resource[AUTHORED_RESOURCE]
a|Reference to owning resource.

h|prop +
  
|*lifecycle_state*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Lifecycle state of the resource, e.g. initial, in_development, published, obsolete.

h|prop +
  0..1
|*custodian_namespace*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Namespace in reverse internet id form of current custodian organisation.

h|prop +
  0..1
|*custodian_organisation*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Plain text name of current custodian organisation.

h|prop +
  0..1
|*copyright*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Optional copyright statement for the resource as a knowledge resource.

h|prop +
  0..1
|*licence*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|Licence of current artefact, in format "short licence name <URL of licence>".

h|prop +
  0..1
|*ip_acknowledgements*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|List of acknowledgements of other IP directly referenced in this archetype,
e.g. terminology codes, keyed by widely known IP source name.

h|prop +
  0..1
|*references*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|List of references of material on which this artefact is based, as keyed strings.

h|prop +
  0..1
|*resource_package_uri*: xref:representation:representation-builtin_types.adoc#_class_string[String]
a|URI of package to which this resource belongs.

h|prop +
  0..1
|*conversion_details*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|Details related to conversion process that generated this model from an original.

h|prop +
  0..1
|*details*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:infostructure:infostructure-resource.adoc#_class_resource_description_item[RESOURCE_DESCRIPTION_ITEM]>
a|Details of all parts of resource description that are natural language-dependent,
keyed by language code.

h|prop +
  0..1
|*other_details*: xref:representation:representation-builtin_types.adoc#_class_hash[Hash]<xref:representation:representation-builtin_types.adoc#_class_string[String],xref:representation:representation-builtin_types.adoc#_class_string[String]>
a|Additional non-language-sensitive resource meta-data, as a list of name/value pairs.
