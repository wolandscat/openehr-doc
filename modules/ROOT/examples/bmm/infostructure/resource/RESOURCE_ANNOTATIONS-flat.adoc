3+^h|  *class RESOURCE_ANNOTATIONS*

h|*Description*
2+a|Object representing annotations on an archetype. These can be of various forms, with a
documentation form defined so far, which has a multi-level tabular structure [ [ [String
value, String key], path key], language key]. Example instance, showing the documentation
structure.

--------
{[
    "en": {[
        "/data[id2]": {[
            "ui": "passthrough"
        ]}
        "/data[id2]/items[id3]": {[
            "design note": "this is a design note on Statement",
            "requirements note": "this is a requirements note on Statement",
            "medline ref": "this is a medline ref on Statement"
        ]}
    ]}
]}
--------
