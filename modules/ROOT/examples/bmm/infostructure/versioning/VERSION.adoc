2+a|Abstract model of one Version within a Version container, containing data,

2+^h|*Feature Group "access"*

  0..1
|*signature*: xref:representation:representation-builtin_types.adoc#_class_string[String]

|*commit_audit*: xref:infostructure:infostructure-versioning.adoc#_class_audit_details[AUDIT_DETAILS]

h| func
|*uid* : xref:representation:representation-identification.adoc#_class_object_version_id[OBJECT_VERSION_ID]
a|Unique identifier of this `VERSION`, in the form of an `{object_id, a version_tree_id,
creating_system_id}` triple, where the `object_id` has the same value as the containing
`VERSIONED_OBJECT _uid`.

h| func
|*preceding_version_uid* : xref:representation:representation-identification.adoc#_class_object_version_id[OBJECT_VERSION_ID]
a|Unique identifier of the version of which this version is a modification; Void if this is

|*data* : T

h| func
|*lifecycle_state* : xref:representation:representation-data_types.adoc#_class_dv_coded_text[DV_CODED_TEXT]
a|Lifecycle state of this version; coded by openEHR vocabulary `version lifecycle state`.

h| func
|*canonical_form* : xref:representation:representation-builtin_types.adoc#_class_string[String]
a|A canonical serial form of this Version, suitable for generating reliable hashes and
signatures. Canonical form of Version object, created by serialising all attributes except
signature.

|*is_branch* : xref:representation:representation-builtin_types.adoc#_class_boolean[Boolean]
