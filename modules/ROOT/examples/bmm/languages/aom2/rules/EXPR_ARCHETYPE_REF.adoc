3+^h|  *class EXPR_ARCHETYPE_REF*

h|*Description*
2+a|Expression tree leaf item representing a reference to a value found in data at a location
specified by a path in the archetype definition.

* A path referring to a value in the archetype (paths with a leading '/' are in the
definition section.
* Paths with no leading '/' are in the outer part of the archetype, e.g.
"archetype_id/value" refers to the String value of the archetype_id attribute of the
enclosing archetype.

h|*Inherit*
2+|xref:languages:bel-gen.adoc#_class_expr_value_ref[EXPR_VALUE_REF]

h|

a|The path to the archetype node.
