3+^h|  *class C_DATE_TIME*

h|*Description*
2+a|Constraint on instances representing date/times, i.e. instants in time.

There is no validity flag for `year`, since it must always be by definition mandatory in
order to have a sensible date/time at all. Syntax expressions of instances of this class
include `"YYYY-MM-DDT??:??:??"` (date/time with optional time) and `"YYYY-MMDDTHH:MM:xx"`
(date/time, seconds not allowed).
