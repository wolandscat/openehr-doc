3+^h|  *class C_ATTRIBUTE*

h|*Description*
2+a|Abstract model of constraint on any kind of attribute in a class model.

h|*Inherit*
2+|xref:languages:aom2-gen.adoc#_class_archetype_constraint[ARCHETYPE_CONSTRAINT]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*rm_attribute_name*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Reference model attribute within the enclosing type represented by a `C_OBJECT`.

h|prop +
  0..1
|*existence*: xref:representation:builtin_types-gen.adoc#_class_multiplicity_interval[Multiplicity_interval]
a|Constraint settable on every attribute, regardless of whether it is singular or of a
container type, which indicates whether its target object exists or not (i.e. is mandatory
or not). Only set if it overrides the underlying reference model or parent archetype in
the case of specialised archetypes.

h|prop +
  0..1
|*children*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:languages:aom2-gen.adoc#_class_c_object[C_OBJECT]>
a|Child `C_OBJECT` nodes. Each such node represents a constraint on the type of this
attribute in its reference model. Multiples occur both for multiple items in the case of
container attributes, and alternatives in the case of singular attributes.

h|prop +
  0..1
|*differential_path*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Path to the parent object of this attribute (i.e. doesn't include the name of this
attribute). Used only for attributes in differential form, specialised archetypes. Enables
only the re-defined parts of a specialised archetype to be expressed, at the path where
they occur.

h|prop +
  0..1
|*cardinality*: xref:representation:builtin_types-gen.adoc#_class_cardinality[Cardinality]
a|Cardinality constraint of attribute, if a container attribute.

h|prop +
  
|*is_multiple*: xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|Flag indicating whether this attribute constraint is on a container (i.e. multiply-valued)
attribute.

h| func
|*any_allowed* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = children.is_empty ∧ ¬ is_prohibited;

a|True if there is no effective constraint on the children of the RM attribute to which this
`C_ATTRIBUTE` refers.

h| func
|*is_mandatory* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = ∃ existence ∧ existence.is_mandatory;

a|True if this `C_ATTRIBUTE` has an existence constraint of 1..1, i.e. mandation.

h| func
|*rm_attribute_path* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Path of this attribute with respect to owning `C_OBJECT`, including differential path
where applicable.

h| func
|*is_single* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True if this node logically represents a single-valued attribute. Evaluated as not
`is_multiple`.

h| func
|*c_congruent_to* ( +
other: xref:languages:aom2-gen.adoc#_class_archetype_constraint[ARCHETYPE_CONSTRAINT]): xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = ¬ ∃ existence ∧ ((is_single ∧ other.is_single) ∨ (is_multiple ∧ other.is_multiple ∧ ¬ ∃ cardinality));

a|True if constraints represented by this node contain no further redefinitions with respect
to the node other, with the exception of `node_id` redefinition in `C_OBJECT` nodes.

Typically used to test if an inherited node locally contains any constraints.

h| func
|*c_conforms_to* ( +
other: xref:languages:aom2-gen.adoc#_class_archetype_constraint[ARCHETYPE_CONSTRAINT], +
rmcc: (String, String) -> Boolean): xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = existence_conforms_to(other) ∧ ((is_single ∧ other.is_single) ∨ (is_multiple ∧ cardinality_conforms_to(other)));

a|True if constraints represented by this node, ignoring any sub-parts, are narrower or the
same as other. Typically used during validation of specialised archetype nodes.

h| func
|*is_prohibited* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = ∃ existence ∧ existence.is_prohibited;

a|True if this `C_ATTRIBUTE` has an existence constraint of `0..0`, i.e. prohibition.

|*existence_conforms_to* ( +
other: xref:languages:aom2-gen.adoc#_class_c_attribute[C_ATTRIBUTE]): xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True if the existence of this node conforms to other.existence

h| func

a|True if the cardinality of this node conforms to other.cardinality, if it exists
