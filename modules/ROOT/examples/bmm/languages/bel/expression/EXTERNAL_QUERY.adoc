3+^h|  *class EXTERNAL_QUERY*

h|*Description*
2+a|Definition of a variable whose value is derived from a query run on a data context in the
operational environment. Typical uses of this kind of variable are to obtain values like
the patient date of birth, sex, weight, and so on. It could also be used to obtain items
from a knowledge context, such as a drug database.

h|*Inherit*
2+|xref:languages:bel-gen.adoc#_class_expr_value[EXPR_VALUE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*context*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Optional name of context. This allows a basic separation of query types to be done in more
sophisticated environments. Possible values might be "patient", "medications" and so on.
Not yet standardised.

h|prop +
  
|*query_id*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Identifier of query in the external context, e.g. "date_of_birth". Not yet standardised.

h|prop +
  0..1
|*query_args*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|Optional arguments to query. Not yet standardised.
