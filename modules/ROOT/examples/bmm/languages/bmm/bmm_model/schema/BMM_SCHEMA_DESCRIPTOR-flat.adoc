3+^h| abstract  *class BMM_SCHEMA_DESCRIPTOR*

h|*Description*
2+a|Descriptor for a BMM schema. Contains a meta-data table of attributes obtained from a
mini-ODIN parse of the schema file.

h|*Inherit*
2+|xref:representation:builtin_types-gen.adoc#_class_any[Any]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*schema_id*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Schema id, formed by the meta-data items:
  <model_publisher>_<schema_name>_<model_release>
e.g. `openehr_rm_1.0.3`, `openehr_test_1.0.1`, `iso_13606_1_2008_2.1.2`.

h|prop +
  
|*meta_data*: xref:representation:builtin_types-gen.adoc#_class_map[Map]<xref:representation:builtin_types-gen.adoc#_class_string[String],xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|Table of `{key, value}` of schema meta-data, keys are string values defined by
`BMM_DEFINITIONS.Metadata_*` constants.

h|prop +
  0..1
|*bmm_schema*: xref:languages:bmm-gen.adoc#_class_bmm_schema[BMM_SCHEMA]
a|Persistent form of model.

h|prop +
  0..1
|*bmm_model*: xref:languages:bmm-gen.adoc#_class_bmm_model[BMM_MODEL]
a|Computable form of model.

h|prop +
  0..1
|*includes*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|Identifiers of schemas included by this schema.

h| func
|*is_top_level* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True if this is a top-level schema, i.e. is the root schema of a 'model'.

h| func
|*is_bmm_compatible* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True if the BMM version found in the schema (or assumed, if none) is compatible with that
in this software.

h|*Invariants*
2+a|

h|
2+a|result_valid: schema_id = BMM_DEFINITIONS.create_schema_id(meta_data.item(BMM_DEFINITIONS.Metadata_model_publisher), meta_data.item(BMM_DEFINITIONS.Metadata_schema_name), meta_data.item(BMM_DEFINITIONS.Metadata_model_release));
