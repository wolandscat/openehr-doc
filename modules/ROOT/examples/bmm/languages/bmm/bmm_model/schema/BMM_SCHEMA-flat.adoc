h|*Description*
2+a|Abstract parent of any persistable form of a BMM model, e.g. `P_BMM_SCHEMA`.

h|*Inherit*
2+|xref:languages:bmm-gen.adoc#_class_bmm_model_metadata[BMM_MODEL_METADATA]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*bmm_version*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Version of BMM model, enabling schema evolution reasoning. Persisted attribute.

h|prop +
  
|*schema_name*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Name of model expressed in schema; a 'schema' usually contains all of the packages of one
'model' of a publisher. A publisher with more than one model can have multiple schemas.

h|prop +
  
|*schema_revision*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Revision of schema.

h|prop +
  
|*schema_lifecycle_state*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Schema development lifecycle state.

h|prop +
  
|*schema_author*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Primary author of schema.

h|prop +
  
|*schema_description*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Description of schema.

h|prop +
  
|*state*: xref:languages:bmm-gen.adoc#_class_bmm_schema_state[BMM_SCHEMA_STATE]
a|Current processing state.

h|prop +
  0..1
|*model_name*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Name of this model, if this schema is a model root point. Not set for sub-schemas that are
not considered models on their own.

h|prop +
  0..1
|*includes*: xref:representation:builtin_types-gen.adoc#_class_map[Map]<xref:representation:builtin_types-gen.adoc#_class_string[String],xref:languages:bmm-gen.adoc#_class_bmm_include_spec[BMM_INCLUDE_SPEC]>
a|Inclusion list of any form of BMM model, in the form of a hash of individual include
specifications, each of which at least specifies the id of another schema, and may specify
a namespace via which types from the included schemas are known in this schema. Persisted
attribute.

h|prop +
  0..1
|*schema_contributors*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>

h|prop +
  0..1
|*bmm_model*: xref:languages:bmm-gen.adoc#_class_bmm_model[BMM_MODEL]
a|Generated by `create_bmm_model` from persisted elements.

h| func
|*schema_id* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Identifier of this schema, used for stating inclusions and identifying files. Formed as:

`{BMM_DEFINITIONS}.create_schema_id ( _rm_publisher_, _schema_name_, _rm_release_)`

E.g. `"openehr_rm_ehr_1.0.4"`.

h| func
|*read_to_validate* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: state = State_includes_processed;

a|True when validation may be commenced.

h|prop +
  
|*rm_publisher*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Publisher of model expressed in the schema.

h|prop +
  
|*rm_release*: xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Release of model expressed in the schema as a 3-part numeric, e.g. "3.1.0" .
