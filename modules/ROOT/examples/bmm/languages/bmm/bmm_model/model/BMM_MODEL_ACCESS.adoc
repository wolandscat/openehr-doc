3+^h|  *class BMM_MODEL_ACCESS*

h|*Description*
2+a|Access to BMM models that have been loaded and validated from one or more schema sets.

h|*Inherit*
2+|xref:representation:builtin_types-gen.adoc#_class_any[Any]

h|
2+^h|*Feature Group "access"*

h|prop +
  0..1
|*schema_directories*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|List of directories where all the schemas loaded here are found.

h|prop +
  0..1
|*all_schemas*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|All schemas found and loaded from `schema_directory`. Keyed by `schema_id`.

h|prop +
  0..1
|*bmm_models*: xref:representation:builtin_types-gen.adoc#_class_map[Map]<xref:representation:builtin_types-gen.adoc#_class_string[String],xref:languages:bmm-gen.adoc#_class_bmm_model[BMM_MODEL]>
a|Top-level (root) models in use, keyed by `model_id`.

h|prop +
  0..1
|*matching_bmm_models*: xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]>
a|Validated models, keyed by `model_id()` and any shorter forms of id, with some or no
versioning information. For example, the keys `"openEHR_EHR_1.0.4"`, `"openEHR_EHR_1.0"`,
`"openEHR_EHR_1"`, and `"openEHR_EHR"` will all match the `"openEHR_EHR_1.0.4"` model,
assuming it is the most recent version available.

h| func
|*bmm_model* ( +
a_model_key: xref:representation:builtin_types-gen.adoc#_class_string[String]): xref:languages:bmm-gen.adoc#_class_bmm_model[BMM_MODEL]
a|Return model containing the model key which is a `model_id` or any shorter form e.g.
model id minus the version. If a shorter key is used, the `BMM_MODEL` with the most recent
version will be selected. Uses `matching_bmm_models` table to find matches if partial
version information is supplied in key.

h| func
|*has_bmm_model* ( +
a_model_key: xref:representation:builtin_types-gen.adoc#_class_string[String]): xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True if a model for a `model_key` is available. A model key is a `model_id` or any
shorter form e.g. model id minus the version. If a shorter key is used, the Result s True
if a `BMM_MODEL` with any version exists.
