3+^h|  *class BMM_CONTAINER_TYPE*

h|*Description*
2+a|Meta-type that specifies linear containers with a generic parameter corresponding to the
type of contained item, and whose container type is a generic type such as `List<T>`,
`Set<T>` etc.

h|*Inherit*
2+|xref:languages:bmm-gen.adoc#_class_bmm_type[BMM_TYPE]

h|
2+^h|*Feature Group "access"*

h|prop +
  
|*container_class*: xref:languages:bmm-gen.adoc#_class_bmm_generic_class[BMM_GENERIC_CLASS]
a|The type of the container. This converts to the `root_type` in `BMM_GENERIC_TYPE`.

h|prop +
  
|*item_type*: xref:languages:bmm-gen.adoc#_class_bmm_unitary_type[BMM_UNITARY_TYPE]
a|The container item type.

h|prop +
  0..1
|*is_ordered*: xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True indicates that order of the items in the container attribute is considered
significant and must be preserved, e.g. across sessions, serialisation, deserialisation
etc. Otherwise known as 'list' semantics.

h|prop +
  0..1
|*is_unique*: xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean]
a|True indicates that only unique instances of items in the container are allowed. Otherwise
known as 'set' semantics.

h| func
|*type_name* : xref:representation:builtin_types-gen.adoc#_class_string[String]
a|Return full type name, e.g. `List<ELEMENT>`.

h| func
|*is_abstract* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = container_type.is_abstract;

a|True if the container class is abstract.

h| func
|*flattened_type_list* : xref:representation:builtin_types-gen.adoc#_class_list[List]<xref:representation:builtin_types-gen.adoc#_class_string[String]> +
 +
  *post_cond* +
    valid_result: Result = item_type.flattened_type_list;

a|Flattened list of type names of `item_type`, i.e. `item_type.flattened_type_list_()`.

h| func
|*unitary_type* : xref:languages:bmm-gen.adoc#_class_bmm_unitary_type[BMM_UNITARY_TYPE]
a|Return `item_type`.

h| func
|*is_primitive* : xref:representation:builtin_types-gen.adoc#_class_boolean[Boolean] +
 +
  *post_cond* +
    valid_result: Result = item_type.is_primitive;

a|True if `item_type` is primitive.

h| func
|*effective_type* : xref:languages:bmm-gen.adoc#_class_bmm_effective_type[BMM_EFFECTIVE_TYPE]
a|Return `item_type.effective_type_()`.
